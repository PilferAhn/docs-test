# Apache Software Foundation — 오픈소스 소프트웨어 재단

> 설립: 1999년 | 본부: 미국 델라웨어 주 | 비영리 단체

---

## 한 줄 정의

**전 세계 개발자들이 함께 만드는 오픈소스 소프트웨어 프로젝트들을 관리하는 비영리 재단**

---

## Apache란?

Apache Software Foundation(ASF)은 오픈소스 소프트웨어를 개발하고 유지하는 비영리 단체입니다.
누구나 무료로 사용할 수 있는 소프트웨어를 만들고, 전 세계 개발자 커뮤니티가 함께 개선합니다.

- **라이선스:** Apache License 2.0 — 상업적 사용 포함 무료
- **운영 방식:** 자원봉사 개발자(Committer)들이 프로젝트를 주도
- **프로젝트 수:** 350개 이상 (2024년 기준)

---

## Apache 프로젝트란?

ASF의 심사를 통과한 오픈소스 프로젝트를 **Apache 프로젝트**라고 부르며, 이름 앞에 "Apache"를 붙입니다.

```
Apache Hadoop
Apache Spark
Apache Airflow
Apache Kafka
Apache Hive
...
```

### 프로젝트가 되는 과정

```
아이디어/외부 프로젝트
      ↓
  Incubator (인큐베이터)
  — 커뮤니티, 거버넌스, 라이선스 검증
      ↓
  Top-Level Project (TLP)
  — 정식 Apache 프로젝트로 졸업
```

## 빅데이터 생태계의 Apache 프로젝트

Apache 프로젝트는 특히 **빅데이터 분야**에서 사실상 표준을 이루고 있습니다.

### 저장

| 프로젝트 | 역할 |
|:---|:---|
| **Hadoop (HDFS)** | 분산 파일 시스템 |
| **Kudu** | 컬럼형 스토리지 |
| **HBase** | NoSQL 분산 데이터베이스 |

### 처리

| 프로젝트 | 역할 |
|:---|:---|
| **MapReduce** | 배치 분산 처리 (Hadoop 구성요소, 별도 프로젝트 아님) |
| **Spark** | 인메모리 분산 처리 |
| **Flink** | 실시간 스트림 처리 |
| **Tez** | DAG 기반 처리 엔진 (Hive 최적화) |

### 쿼리

| 프로젝트 | 역할 |
|:---|:---|
| **Hive** | SQL on Hadoop |
| **Impala** | 실시간 SQL (Cloudera) |

### 메시징 / 파이프라인

| 프로젝트 | 역할 |
|:---|:---|
| **Kafka** | 분산 메시지 스트리밍 |
| **NiFi** | 데이터 플로우 자동화 |

### 오케스트레이션 / 관리

| 프로젝트 | 역할 |
|:---|:---|
| **Airflow** | 워크플로우 스케줄링 |
| **Oozie** | Hadoop 워크플로우 (구세대) |
| **ZooKeeper** | 분산 시스템 코디네이션 |
| **YARN** | 클러스터 리소스 관리 |

## 이 폴더 구성

```
apache/
├── airflow/      — 워크플로우 오케스트레이션
├── spark/        — 인메모리 분산 처리
└── hadoop/       — 분산 저장 및 처리 플랫폼
    └── mapreduce/ — 배치 분산 처리 프레임워크
```
