# vi 명령어 모음집

---

## 모드 전환

```
i        현재 커서 위치에서 입력 모드
a        커서 다음 위치에서 입력 모드
o        현재 줄 아래 새 줄 만들고 입력 모드
O        현재 줄 위에 새 줄 만들고 입력 모드
ESC      입력 모드 → 명령 모드
```

---

## 저장 / 종료

```
:w       저장
:q       종료 (변경사항 없을 때)
:wq      저장 후 종료
:q!      저장 안 하고 강제 종료
:wq!     강제 저장 후 종료
ZZ       저장 후 종료 (단축키)
```

---

## 이동

```
h        왼쪽
l        오른쪽
j        아래
k        위

w        다음 단어 시작
b        이전 단어 시작
e        현재 단어 끝

0        줄 맨 앞
$        줄 맨 끝
^        줄 첫 번째 문자 (공백 제외)

gg       파일 맨 위
G        파일 맨 아래
:n       n번째 줄로 이동  (예: :30)

Ctrl+d   반 페이지 아래
Ctrl+u   반 페이지 위
Ctrl+f   한 페이지 아래
Ctrl+b   한 페이지 위
```

---

## 편집

```
x        커서 위치 문자 삭제
dd       현재 줄 삭제 (잘라내기)
D        커서부터 줄 끝까지 삭제
dw       커서부터 단어 끝까지 삭제
d$       커서부터 줄 끝까지 삭제

yy       현재 줄 복사
yw       단어 복사
y$       커서부터 줄 끝까지 복사

p        커서 아래에 붙여넣기
P        커서 위에 붙여넣기

u        실행 취소 (undo)
Ctrl+r   다시 실행 (redo)

r        커서 위치 문자 하나 교체
cw       단어 교체 (삭제 후 입력 모드)
cc       줄 전체 교체
C        커서부터 줄 끝까지 교체
```

---

## 복수 줄 / 숫자 조합

```
5dd      5줄 삭제
3yy      3줄 복사
10j      10줄 아래로 이동
2w       2단어 앞으로
```

---

## 검색

```
/검색어       앞으로 검색
?검색어       뒤로 검색
n             다음 검색 결과
N             이전 검색 결과
```

---

## 치환

```
:s/old/new         현재 줄 첫 번째 old → new
:s/old/new/g       현재 줄 전체 old → new
:%s/old/new/g      파일 전체 old → new
:%s/old/new/gc     파일 전체 치환 (하나씩 확인)
```

---

## 화면 분할

```
:sp 파일명    수평 분할
:vsp 파일명   수직 분할
Ctrl+w w      다른 창으로 이동
Ctrl+w q      현재 창 닫기
```

---

## 기타

```
.            마지막 명령 반복
:%d          파일 전체 삭제
:set nu      줄 번호 표시
:set nonu    줄 번호 숨기기
:set paste   붙여넣기 모드 (들여쓰기 자동 방지)
```

---

## 자주 쓰는 조합

```
ggdG         파일 전체 삭제 (gg → 맨 위, dG → 끝까지 삭제)
gg=G         파일 전체 들여쓰기 정렬
:%y          파일 전체 복사
:w !sudo tee %   sudo 권한으로 저장 (읽기 전용 파일 수정 시)
```
