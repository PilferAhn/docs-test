# CDF - Cloudera DataFlow (GPT 답변)

> 출처: ChatGPT 답변 원문
> GPT는 CDF를 직접 비교하지 않았습니다.
> GPT 답변은 하둡 인프라 컴포넌트 수준에 집중했기 때문에, CDF와 같은 CDP 상위 스트리밍 서비스는 비교 범위에 포함되지 않았습니다.

## GPT 관련 맥락

GPT 답변에서 CDF와 직접 연관된 내용은 없습니다.
다만, CDF가 기반으로 사용하는 인프라 컴포넌트들은 GPT 답변과 관련됩니다.

- CDF는 YARN 위에서 NiFi/Flink/Kafka를 실행 → GPT의 YARN (`10_YARN/gpt.md`) 참고
- CDF의 데이터는 HDFS에 저장됨 → GPT의 HDFS (`01_HDFS/gpt.md`) 참고

## GPT 관점 요약

CDF의 Databricks 대응에 대한 상세 내용은 Gemini 답변(`gemini.md`) 또는 Claude 정리(`claude.md`)를 참고하세요.
