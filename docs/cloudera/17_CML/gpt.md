# CML - Cloudera Machine Learning (GPT 답변)

> 출처: ChatGPT 답변 원문
> GPT는 CML을 직접 비교하지 않았습니다.
> GPT 답변은 하둡 인프라 컴포넌트 수준(HDFS, YARN, Hive 등)에 집중했기 때문에,
> CML과 같은 CDP 상위 ML 서비스는 비교 범위에 포함되지 않았습니다.

## GPT 관련 맥락

GPT 답변에서 CML과 직접 연관된 내용은 없습니다.
다만, CML이 사용하는 기반 인프라 컴포넌트들은 GPT가 다룬 내용과 관련됩니다.

- CML은 Spark를 데이터 처리에 활용 → GPT의 YARN, HDFS 내용과 연관
- CML은 HDFS 기반 데이터에 접근 → GPT의 HDFS (`01_HDFS/gpt.md`) 참고
- CML의 거버넌스는 Ranger가 담당 → GPT의 Ranger (`07_Ranger/gpt.md`) 참고

## GPT 관점 요약

CML의 Databricks 대응에 대한 상세 내용은 Gemini 답변(`gemini.md`) 또는 Claude 정리(`claude.md`)를 참고하세요.
